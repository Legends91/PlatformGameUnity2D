# PlatformGameUnity2D
Câu chuyện xoay quanh một cô gái nhỏ thích phiêu lưu tên Luna bị thất lạc vào một truyền thuyết sống. Đây sẽ là một chuyến phiêu lưu lớn nhất của Luna. Những gì Luna trải nghiệm trên chuyến phiêu lưu này là bạn. Hãy dẫn dắt cô ấy trải nghiệm chuyến phiêu lưu này nhé!
